# Museeka Audio Credits

Museeka uses short public-domain/traditional musical excerpts as source material for the demo score:

- Ludwig van Beethoven, "Ode to Joy" theme, public domain.
- Johann Pachelbel, "Canon in D" ground progression, public domain.
- "Frere Jacques", traditional, public domain.
- J. S. Bach, "Prelude in C Major" inspired arpeggio material, public domain.
- "Greensleeves", traditional, public domain.

Instrument playback uses selected MP3 samples from the `tonejs-instruments` family of packages:

- `tonejs-instrument-piano-mp3`
- `tonejs-instrument-harp-mp3`
- `tonejs-instrument-flute-mp3`
- `tonejs-instrument-xylophone-mp3`
- `tonejs-instrument-bass-electric-mp3`

Package code is MIT licensed. The samples are credited by the upstream `tonejs-instruments` project as CC BY 3.0.
